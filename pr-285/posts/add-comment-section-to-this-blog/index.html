GitHub Issuesをサイトのコメント欄として活用できるutterancesを導入した

2024/03/17 1 min read (356 words)
#blog 

このブログにコメント欄を設置した。

utterance/utterancesというGitHubのIssuesへのコメントを記事のコメント欄として使えるようにしてくれるOSSのツールを使っている。1

導入に必要だったのは

  1. コメント用のリポジトリ作成2
  2. utterances
    A lightweight commenting system using GitHub issues.
    https://utteranc.es/
    でscriptタグを生成
  3. 生成したタグをブログに埋め込む

だけだったので10分くらいでできて大変お手軽だった。

(自分はこのブログのリポジトリのissueをコメント用に使いたくなかったのでそれ用のリポジトリを作成したが、もし既存のリポジトリを使う場合は1.は不要)

ぜひご自由にコメントを残していってください。
