hugo-theme-terminal でOGPに任意の画像を設定する方法

2023-06-10 :: kyu08 :: 1 min read (251 words)
#hugo  #blog 

目次

  • 前提
  • 困り
  • やったこと

hugo-theme-terminal でOGPの設定に少し詰まったのでやったことを書いておく。

前提⌗

  • hugo version: hugo v0.112.5+extended darwin/arm64

困り⌗

  • 筆者はカバー画像を記事一覧・記事詳細画面に表示したくない、かつOGPは全共通の画像を使いたい
  • hugo-theme-terminal では任意の画像をOGPだけに設定することができない。（カバー画像をOGPとすることはできる）

やったこと⌗

多少ハッキーな方法ではあるが以下の方法で実現した。

  • 各記事のFront Matterにcover: "${OGP_URL}"指定(archetypesにテンプレートとして書いておくと便利)
  • 以下のCSSを追加
.post-cover {
  display: none;
}
